Install (2 minutes)

1

Make sure you have Python 3.8+

Check by running:

python3 --version

If you don't have it, download from python.org. Windows users: try python --version instead.

2

Download and install

Open your terminal and run these 4 commands:

git clone https://github.com/MSFdoe/buildfirst.git cd buildfirst python3 -m venv .venv .venv/bin/pip install -r requirements.txt
git clone https://github.com/MSFdoe/buildfirst.git cd buildfirst python -m venv .venv .venv\Scripts\pip install -r requirements.txt
3

Run setup (pick your AI provider)

.venv/bin/python -m buildfirst setup
.venv\Scripts\python -m buildfirst setup

You'll pick from 4 AI providers:

Ollama Free, private, local
Claude ~$0.05 per guide
ChatGPT ~$0.05 per guide
Gemini Free tier available

Not sure? Start with Ollama — it's free and nothing leaves your computer.

4

Generate your first guide

Point it at any project folder:

.venv/bin/python -m buildfirst compile ~/my-project --title "My Project"
.venv\Scripts\python -m buildfirst compile C:\my-project --title "My Project"

Takes 1–2 minutes. You'll get a single .md file with all 9 sections.

That's it. Every guide follows the same 9-part structure: Purpose, Table of Contents, Prerequisites, Decision Table, Quick Start, Tutorial (Concept → Code → Trace), Worked Trace, Glossary, and Session Log. Point it at any folder and get a complete learning guide.
Updates. To get the latest version, open your terminal, go to the buildfirst folder, and run git pull. That's it.
